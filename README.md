## Peer to Peer Video Chat Web App

#### Client

Packages:

- React
- WebRTC Adapter
- Socket.io Client

Do before build:

- define firebase API key as REACT_APP_PEER_FIREBASE environemnt variable in .env file as JSON string

Commands:

- npm run start

#### Sever

Packages:

- Node
- Express
- Socket.io

Hosting:

- Firebase

Commands:

- npm run serve
